Programming Concepts

Because Visual Basic for application (VBA) is a real, live programming language, it uses many elements common to all programming languages. In subsequent topics, I introduce you to several of these elements:

  • Comments
  • Variables
  • Constants
  • Data types
  • Arrays and few other concepts.

If you’ve programmed with other languages, some of this topics will be familiar. If you’re a programming newbie, it’s time to roll up your sleeves and get busy.
