About website

CADwCODE is a developer website, with tutorials on development languages such as Visual Basic for Application (VBA), and C#. Basically, this website is a place where I put all my CAD Macros & its API related knowledge for public share.

This website is for any working professionals who want to learn VBA/macro programming to automate their regular repeated work, college going students who want to enhance their CAD knowledge base and to some extent CAD Software Developers who wants to learn some knowledge upon CAD Development. Hence, this website is for those people who want to learn VBA macros & if interested creating Add-ins for those CAD tools from scratch with no programming knowledge.

Since by education I am not a computer programmer and I don’t know about “Article writing”, hence all article written here used languages from different material such as SOLIDWORKS API HELP and other programming materials. Hope you ignore any typo & grammatical errors and learn from the website.

About myself

While doing some Add-in related projects I worked on Excel programming. Since I enjoy programming of SOLIDWORKS & MS Excel I thought why don’t share my knowledge to everyone.

If this website helps anyone I think purpose of this website is fulfilled.
